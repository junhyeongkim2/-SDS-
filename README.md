## DFS Depth First Search 깊이 우선 탐색

그래프 탐색 방법의 한가지

한 경로로 탐색하다 특정상황에서 최대한 깊숙하게 들어가서 확인 후 다시 돌아가 다른 경로로 탐색하는 방식

구조상 stack overflow에 유의

그래프 알고리즘에서 기초가 되는 탐색방식으로 반드시 숙지가 필요함
